# Currency Converter Utility

Консольное приложение на C# для конвертации валют в реальном времени на основе заданных коэффициентов.

### Технический стек
* Платформа: .NET 6+
* Язык: C#
* Структуры данных: Dictionary<string, decimal> для хранения курсов.

### Реализованный функционал
* Валидация пользовательского ввода (decimal.TryParse).
* Динамический расчет кросс-курсов для нескольких валютных пар (USD, EUR, CNY).
* Обработка точности финансовых вычислений через тип decimal.
